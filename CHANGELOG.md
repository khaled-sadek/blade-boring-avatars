# Changelog

All notable changes to `blade-boring-avatars` will be documented in this file.

## [2.0.0] - 2025-09-25

### Added
- GitHub Actions workflow for automatic releases to Packagist
- Badges for Packagist version, downloads, and license in README

### Changed
- Updated minimum PHP version to 8.2
- Updated Laravel support to 10.x, 11.x, and 12.x
- Updated development dependencies

### Removed
- Support for PHP 8.1 and below
- Support for Laravel 9.x and below

## [1.0.0] - 2023-01-01

Initial release of the package.
